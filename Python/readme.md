# Bank Customer Churn Predition Project
This project is  based on cutomer churn data of a bank that was downloaded from kaggle, the data is provided in the churn_data.csv file, it has data for 28,383 customers and has information regarding 21 variables.
The python code is provided in bank_customer-churn_prediction.ipynb file. Several Data mining procedures were performed to understand the data and preprocess the data for it to be ready to build machine learning models that can predicts the churn possibilities for fiture customers.
